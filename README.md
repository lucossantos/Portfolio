## 🚀 Sobre o projeto

O objetivo deste portfólio é reunir projetos e experiências em um único lugar, utilizando uma interface moderna, responsiva e simples de navegar.

O site foi desenvolvido com foco em:

- Organização visual
- Responsividade
- Navegação intuitiva
- Exposição de projetos
- Prática de desenvolvimento front-end
