Gerador de Senhas Aleatórias

Este projeto gera senhas aleatórias e seguras utilizando letras, números e símbolos personalizados pelo usuário. A versão atual permite escolher os tipos de caracteres desejados, copiar a senha com um clique e utiliza crypto.getRandomValues() para uma geração mais segura e profissional. O projeto foi desenvolvido originalmente em Python e posteriormente adaptado para HTML, CSS e JavaScript para integração ao portfólio. Projeto em Python no GitHub
