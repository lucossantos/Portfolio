Gerador de Senhas Aleatórias

Este código define a função 'generate_password', que cria senhas aleatórias com letras, números e símbolos, usando os módulos 'string' e 'random'.
Por padrão, gera uma senha de 12 caracteres, mas o tamanho pode ser ajustado.
O projeto foi feito em Python e depois adaptado para HTML com JavaScript para uso no portfólio.
